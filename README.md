# dotgpg
> Ian's GPG configuration (part of [dotfiles][dotfilesUrl])

## Installation

Copies GPG configuration to your home directory:

```console
./install.sh
```

## Update

Copies the GPG configuration in your home directory back to the repo:

```console
./update.sh
```
